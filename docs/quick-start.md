# 快速开始指南

## 适用范围

本指南适用于当前版本的 `tpLogTool`，帮助首次使用者快速完成安装、配置和常用功能体验。

## 首次使用

程序不再内置默认账号密码，首次使用必须先配置账号密码。

### 配置账号密码

1. 启动程序
2. 点击菜单栏右侧的设置按钮
3. 在弹出的设置对话框中：
    - 运维账号：用于设备、记录等运维查询功能
    - 固件账号：用于固件管理功能
    - 可按需配置一个或两个账号
4. 输入账号和密码
5. 点击测试连接验证账号（可选）
6. 点击保存
7. 开始使用程序

### 自动提示

如果未配置账号密码，在使用对应功能时会自动弹出提示：
- 运维功能：提示前往配置运维账号
- 固件功能：提示前往配置固件账号
- 点击OK会自动打开设置对话框并切换到对应标签页
- 配置完成后即可开始使用

## 配置说明

### 配置存储

配置保存在Windows注册表中：
```
HKEY_CURRENT_USER\Software\TPQueryTool
```

### 配置优先级

```
注册表配置（必须配置）
```

## 常见场景

### 场景1：首次使用

**操作**：
1. 启动程序
2. 输入设备SN或ID
3. 点击"查询"按钮
4. 系统提示配置账号密码
5. 点击"是"打开设置对话框
6. 输入账号密码并保存
7. 重新点击"查询"按钮

### 场景2：更换账号

**操作**：
1. 点击"⚙️"图标按钮
2. 切换到对应账号标签页
3. 修改配置
4. 点击测试连接（可选）
5. 点击保存

### 场景3：多人使用同一台电脑

**说明**：
- 每个Windows用户登录后配置自己的账号
- 配置会自动隔离，互不影响

## 快速测试

### 测试配置是否生效

1. 打开设置界面
2. 切换到"账号配置"标签页
3. 点击对应账号组的"验证"按钮
4. 查看提示信息：
   - ✅ "运维账号验证成功" / "固件账号验证成功" → 配置正确
   - ❌ "连接失败" → 检查账号密码

## 注意事项

1. **首次使用**：必须先配置账号密码
2. **独立配置**：运维账号和固件账号可以独立配置
3. **配置保存**：配置会永久保存，下次启动自动使用
4. **密码安全**：不要在共享电脑上保存敏感账号
5. **验证按钮**：保存前建议先验证连接
6. **固定环境**：程序固定使用生产环境
7. **自动提示**：未配置时使用对应功能会自动提示配置
8. **记录页限制**：无筛选条件时不允许直接查询
