# 快速开始指南

## 新功能：账号密码配置

从 v1.1.0 版本开始，程序支持自定义账号密码配置。

## 首次使用

程序不再内置默认账号密码，首次使用必须先配置账号密码。

### 配置账号密码

1. 启动程序
2. 点击菜单栏右侧的 **"⚙️"图标按钮**
3. 在弹出的对话框中：
   - 输入账号
   - 输入密码
4. 点击 **"测试连接"** 验证账号（可选）
5. 点击 **"保存"** 按钮
6. 开始使用程序

### 自动提示

如果未配置账号密码，在查询时会自动弹出提示：
- 提示"检测到账号密码未配置，是否现在配置？"
- 点击"是"会自动打开设置对话框
- 配置完成后即可开始查询

## 配置说明

### 配置存储

配置保存在Windows注册表中：
```
HKEY_CURRENT_USER\Software\TPQueryTool
```

### 配置优先级

```
注册表配置（必须配置）
```

## 常见场景

### 场景1：首次使用

**操作**：
1. 启动程序
2. 输入设备SN或ID
3. 点击"查询"按钮
4. 系统提示配置账号密码
5. 点击"是"打开设置对话框
6. 输入账号密码并保存
7. 重新点击"查询"按钮

### 场景2：更换账号

**操作**：
1. 点击"⚙️"图标按钮
2. 输入新的账号密码
3. 点击"测试连接"验证
4. 点击"保存"

### 场景3：多人使用同一台电脑

**说明**：
- 每个Windows用户登录后配置自己的账号
- 配置会自动隔离，互不影响

## 快速测试

### 测试配置是否生效

1. 打开设置界面
2. 点击"测试连接"按钮
3. 查看提示信息：
   - ✅ "连接成功" → 配置正确
   - ❌ "连接失败" → 检查账号密码

### 测试查询功能

1. 在"状态"页面输入设备SN：
   ```
   TEST123456
   ```
2. 点击"查询"按钮
3. 查看查询结果

## 注意事项

1. **首次使用**：必须先配置账号密码
2. **配置保存**：配置会永久保存，下次启动自动使用
3. **密码安全**：不要在共享电脑上保存敏感账号
4. **测试连接**：保存前建议先测试连接
5. **固定环境**：程序固定使用生产环境
6. **自动提示**：未配置时查询会自动提示配置
