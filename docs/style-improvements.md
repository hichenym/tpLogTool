# 样式改进指南 - v2.0.0

## 问题描述
1. 下拉框下拉箭头不明显，用户难以识别
2. 应用中的强调色（青绿色 #0d7377）过于鲜艳和突兀

## 解决方案

### 颜色调整方案

将所有过于鲜艳的青绿色 `#0d7377` 改为柔和的灰色 `#505050`，使整个应用风格更加统一和舒适。

| 组件 | 旧颜色 | 新颜色 | 说明 |
|------|--------|--------|------|
| 下拉箭头背景 | #3c3c3c | #505050 | 浅灰，适度对比 |
| 选中项背景 | #0d7377 | #505050 | 柔和灰色 |
| 输入框选中文字 | #0d7377 | #505050 | 柔和灰色 |
| 按钮按下状态 | #0d7377 | #505050 | 柔和灰色 |
| 复选框选中状态 | #0d7377 | #505050 | 柔和灰色 |
| 滚动条悬停 | #0d7377 | #6a6a6a | 更浅的灰色 |
| 菜单按钮选中 | #0d7377 | #505050 | 柔和灰色 |
| 分隔器悬停 | #0d7377 | #6a6a6a | 更浅的灰色 |

### 改进内容

#### 1. 下拉框样式
- 下拉箭头背景: `#505050`（浅灰）
- 下拉箭头宽度: 24px（便于点击）
- 下拉箭头左边框: `#555555`（与主框一致）
- 移除右边距: 消除右边空隙色差
- 选中项背景: `#505050`（柔和灰色）
- 悬停边框: `#555555`（保持灰色）

#### 2. 输入框样式
- 选中文字背景: `#505050`（柔和灰色）
- 保持原有的边框和背景

#### 3. 按钮样式
- 悬停状态: `#4a4a4a`（保持原有）
- 按下状态: `#505050`（柔和灰色）
- 禁用状态: `#2b2b2b`（保持原有）

#### 4. 复选框样式
- 选中状态: `#505050`（柔和灰色）
- 悬停状态: 保持灰色边框
- 禁用状态: `#3c3c3c`（保持原有）

#### 5. 滚动条样式
- 悬停状态: `#6a6a6a`（更浅的灰色）
- 正常状态: `#555555`（保持原有）

### 修改文件

- ✅ `query_tool/main.py` - 全局样式表
- ✅ `query_tool/utils/style_manager.py` - 样式常量
- ✅ `query_tool/pages/gitlab_log_page.py` - 页面特定样式

### 受影响的页面

1. **设备页面** - 无下拉框
2. **GitLab 日志页面** - 项目、分支、提交者下拉框
3. **账号查询页面** - 型号下拉框
